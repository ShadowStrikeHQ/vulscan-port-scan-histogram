# vulscan-port-scan-histogram
Generates a histogram of open ports across a given IP range.

## Install
`git clone https://github.com/ShadowStrikeHQ/vulscan-port-scan-histogram`

## Usage
`./vulscan-port-scan-histogram [params]`

## Parameters
- `-ip_range`: IP range to scan.
- `-port_list`: Comma-separated list of ports to scan.
